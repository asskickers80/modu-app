# ORDER-landlord-home-align-v1 — 임대인 홈 골격 통일 (양도인 기준)

## 모드: 자율 판정. 멈춤 (a)(b)(c)만.

## 결정 (대표 확정)
임대인 홈을 양도인 홈의 검증된 골격 순서로 통일:
① 인사+헤더 → ② 내 상가 카드 → ③ 진행 가이드 → ④ 임대 현황 요약(매출 카드 대응)
→ ⑤ 상가 지표·문의 알림 → ⑥ 오늘의 한 마디 → ⑦ 완성도

## 항목별 지시
① 헤더: 진실의 원천 — 실등록 상가 기준(지역·상태), 미등록 시 A3 폴백.
   헤드라인: "총 상가 N · 공실 N · 매각 N · 임대 중 N", 1개면 단수 표기.
② 내 상가 카드: 0건=등록 CTA / 1+=세로 쌓기, 최대 3개, 4개째부터 "외 N개" 접힘.
   양도인 매물 카드 동형(썸네일·상태 뱃지·핵심 숫자·탭→E2L). + 새 상가 등록 링크.
③ 가이드: 양도인 가이드 구조 재사용(관찰 가능·전단계 탭·문의받기/협의시작 2칸).
   단계는 임대인 실동작 기준 자율 판정, 관찰 불가 단계 금지.
④ 임대 현황 요약: 신설 자리. 실데이터 소스 없음 → 정직 "준비중" 카드. 더미 금지.
⑤ 지표·문의: 양도인 구조 그대로 재사용(새 문의 레드·펄스·UnreadDot + 전체 서브, 진척도 준비중). 복제 금지.
⑥ 오늘의 한 마디: Gemini 배치 재사용, 임대인 문구 배치 없으면 "준비중" + 배치 확장 필요 보고.
⑦ 완성도: 골격만 "준비중". calcScoreLandlord 스텁만, 미구현 명시.

## 원칙
공유 가능 컴포넌트 추출·재사용(복제 금지). 모두 화법, 권위 어휘·AI 라벨 금지.

## 테스트: 0/1/3/5건(접힘) 카드, 가이드 점등, 문의 지표. 전체 스위트.
