## 자율 판정 근거 (요약)
- **#4 기업회원 비적용**: B2B 6유형(중개·시설·세무·금융·가맹·서비스)은 소매 업종(8 대분류) 택소노미와 다른 축이라 IndustryPicker 부적합.
- **#12 창업준비·운영중 "부분→후속"**: 빈점포·프랜차이즈 데모 카드(조회 128·전국 3,400점·가맹비)와 SalesInputPage 더미 매출은 헌법상 "가짜 숫자"이나, 전면 정직화(피드 콘텐츠 교체)는 **제품 방향 결정**이라 데모 피드를 일괄 제거하지 않고 후속 조각으로 분리 권고. (무리한 임의 개편 회피)
- **#15 전제 불일치(정직 공개)**: 오더는 "운영중·기업회원=사업자번호 가진 축"을 전제했으나, 실제 코드는 사업자번호·폐업 자동비공개가 **양도(seller) 축 전용**이고 운영중은 사업자번호를 아예 수집하지 않으며 기업회원은 인증이 MOCK이라 bizno 미연결. → 두 축 모두 현재 비적용(대상 로직 부재).
- **임대인 유보 [임대인-설계후]**: 1·2·3·4·5(E2L)·6·13(실 DM) — 전부 상가 객체 저장/모델 미확정 의존. 안전 항목(8·12)만 적용.
