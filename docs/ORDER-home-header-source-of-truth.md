# ORDER: 홈 헤더 업종·지역 — 매물 기준 전환

## 배경
- 현재 홈 헤더("외식 > 패스트푸드 양도 준비 중 / 경기 지역")는 온보딩 선택값 고정
- 실제 등록 매물(원주시)과 불일치 발생. 온보딩 답변은 임시값이고, 매물 등록 후에는 매물이 진실의 원천

## 요건
1. 본인 매물 0건 (example 제외): 현행 유지 — 온보딩 선택값 표시
2. 매물 1건 이상: 헤더의 업종·지역을 매물 데이터에서 파생
   - 업종: 매물의 대분류 > 소분류
   - 지역: 매물 주소의 시/도 단위 (예: "강원 지역"). 시·군까지는 표시하지 않음 (기존 상호 비공개 정책과 톤 일치)
   - 2건 이상이면 가장 최근 등록(또는 최근 수정) 매물 기준
3. 온보딩 원본 값은 DB에 그대로 보존 (덮어쓰지 않음 — 표시 로직만 분기)
4. 매물 수정으로 주소·업종이 바뀌면 헤더도 즉시 반영

## 검증
- Playwright: 0건 → 온보딩값 / 1건(원주 매물) → "강원 지역" / 매물 주소 수정 → 헤더 갱신
- 실기기 확인 후 완료
