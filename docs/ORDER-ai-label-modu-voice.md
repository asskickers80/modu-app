# ORDER: "AI" 라벨 전면 제거 — 모두 화법 통일 (홈 정보 섹션)

## 배경
E1 카피 오더(ORDER-e1-copy-modu-voice)의 "AI 단어 제거, 모두 주어 화법" 원칙이 홈 정보 섹션에 미적용. "✨ AI 맞춤 정보"가 대표 사례.

## 요건
1. 전수 조사: 사용자 노출 문자열에서 "AI" 잔존 지점 grep 전수 스캔 → 목록 보고 (이번에 보인 곳만 고치고 끝내기 금지)
2. 교체 원칙: 기계 주어("AI가") → 모두 주어("모두가 찾아준", "모두가 챙겨온"). 기능별 자연스러운 문구는 재량, 단 "알짜 정보" 컨셉 유지
   - 예: "✨ AI 맞춤 정보" → "모두가 찾아온 알짜 정보"
   - ✨ 등 AI 연출 이모지도 함께 제거 (섹션 이모지는 기존 📈 같은 내용 기반만 유지)
3. 동종 시장 동향 우측 라벨 간소화: "요식업 > 피자·버거·샌드위치 최신 뉴스" → 소분류만 "피자·버거·샌드위치" (업종 재고지 불필요, 390px 공간 확보)
4. '오늘의 한 마디' AI 뱃지도 이번에 함께 정리 (기존 대기 안건) — 뱃지 제거 또는 모두 화법 문구로
5. 코드 내부 식별자·주석은 대상 아님. 사용자 노출 문자열만

## 검증
- grep 증빙: 교체 전 목록 vs 교체 후 사용자 노출 "AI" 0건
- Playwright: 홈 섹션 라벨 렌더
- 실기기 확인 후 완료
