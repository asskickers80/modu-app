# ORDER-landlord-content-v1 — 임대인 콘텐츠 블록 일괄 (뉴스+필독+오늘의 한 마디)

## 모드: 자율 판정. 멈춤 (a)(b)(c)만.

## 배경
홈 골격 통일 때 콘텐츠 블록(필독·시장동향 뉴스)이 누락. 임대인 홈에 양도인 동형으로 채운다.

## 1. 시장동향 뉴스 (네이버 API 재사용)
- 임대인용 부동산 키워드 축 신설(상가 임대차·매매·공실률·상권 동향·상가 투자·임대차보호법). 13업종 축과 별개.
- 홈 배치: 양도인 동형(콘텐츠 영역).
- 의도 가중(공실→임대/매각→매매)이 기존 매칭으로 가벼우면 적용, 구조 변경 필요하면 일괄 노출 단순화 + 근거.
- 0건 준비중 폴백 재사용.

## 2. 임대인 필독 (Gemini 배치 신규)
- 양도자필독 구조 재사용, 주제 임대인(상가임대차보호법·공실 대응·임차인 심사·매각 절차/세금 개요·권리금 회수기회). 법률·세무 개요+"전문가 확인", 모두 화법.
- 5초 지연+지수 백오프. 날짜 폴백 동일.

## 3. 오늘의 한 마디 (landlord 배치)
- daily_contents landlord 코칭 적재(seller 구조 재사용). 모두 화법, 권위 어휘 금지, 불확실하면 침묵.
- 홈 ⑥ 준비중 → 실콘텐츠.

## 4. 공통: AI 라벨 금지(ModuMark), 출처 표기 규격. 적재 후 SQL 카운트 증빙(콘텐츠 배치=승인 패턴).

## 테스트: 뉴스 렌더·0건 폴백, 필독 진입·폴백, 한 마디 표시. 전체.
## 보고: 4줄(ls-remote) + 키워드 세트 + 필독 주제 목록 + 배치 적재 카운트.
