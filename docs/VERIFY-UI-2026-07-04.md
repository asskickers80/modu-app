## 발견 이슈

1. **Daum 첫 클릭 실패**: /e1/1 주소 검색 첫 클릭 시 모달 미오픈 + postcode.v2.js `Uncaught TypeError: Cannot read properties of null (reading 'document')`. 두 번째 클릭에 정상 오픈. 임베드 컨테이너 준비 전 초기화 레이스 의심.
2. **E1/2 에러 헤딩 잔존**: 503 후 "다시 시도" 성공 시 초안은 렌더되나 상단에 "잠깐, 문제가 생겼어요" 문구가 남음 (병렬 시세해석 콜 실패 잔재 추정).
3. **E1/5 사진 체크 불일치**: 사진 1장 업로드인데 "내부 사진 3장" 항목이 완료(초록)로 표시. 라벨과 판정 조건 불일치.
4. (참고) E1/2 실거래 블록의 가격대 표기 "4,300~2,260,000만원" — 이상치·포맷 정리 여지.
5. (참고) SELLER-CHECKLIST.md 일부 항목 구버전: A7 매출 카드 더미(2,840만원)·시장동향 더미·오픈채팅 더미 목록은 현재 "서비스 준비중" 정직 빈 상태로 교체돼 있음 (병행 세션 정리분).
