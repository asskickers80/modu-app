# ORDER: E2 매물 상세 — 소유자 모드

## 배경
홈 내 매물 카드 → E2 상세 진입 시 방문자와 동일하게 "문의하기"가 노출됨. 소유자에게는 수정 진입점이 보여야 함. 레퍼런스(당근마켓): 본인 글 상세는 같은 화면에 하단 액션만 소유자용으로 교체.

## 요건
1. 소유자 판정: 현행 소유 판정 로직(device_id) 재사용. 본인 매물이면 소유자 모드
2. 소유자 모드에서 바뀌는 것:
   - 하단 "문의하기" 제거 → "매물 수정하기" (현행 ?edit= 흐름 진입)
   - 상태 변경 액션 노출: 공개중 ↔ 협의중 등 현행 status 전환 (전환 가능 상태는 현행 정의 따름)
   - 상단에 얇은 안내 바: "내 매물이에요 · 방문자에게 이렇게 보여요" (모두 화법)
3. 소유자 모드에서 유지되는 것: 본문은 방문자와 동일 (공개 모습 확인 목적)
4. 방문자 화면은 변경 없음

## 주의
- 관심/찜 등 방문자 전용 액션이 더 있으면 소유자 모드에서 숨김
- 더보기 시트의 수정 진입과 중복돼도 무방 (진입점 다중화 허용)

## 검증
- Playwright: 소유 매물 → 수정 버튼 + 상태 전환 + 문의하기 부재 / 타인 매물 → 현행 그대로
- 상태 전환 후 홈 카드 뱃지 갱신 확인
- 실기기 확인 후 완료
