# ORDER-inquiry-alert-v2 — 새 문의 타일 시각 강화

1. 숫자 크기: 새 문의 수를 타일 내 최대 요소로 (기존 대비 1.5~2배, 다른 타일 숫자보다 큼).
2. 색: 미확인>0일 때 네이비 대신 경고 계열 강조 (레드/오렌지 톤, 메시지 탭 빨간 점과 동일 계열).
   9색 토큰 내에서 선택, 토큰에 없으면 후보 제시 후 멈춤.
3. 모션: 홈 진입 시 소프트 펄스 2~3회 후 정지 (무한 깜빡임 금지).
   + 타일 우상단 빨간 점 뱃지 (메시지 탭과 동일 컴포넌트 재사용, 복제 금지).
4. 라벨: "새 문의 1"이 주, "전체 3"은 현행 서브 유지.
5. 0건이면 전부 해제 (현행 로직 유지).
6. Playwright: 미확인>0 강조·뱃지 표시 / 0건 해제. 펄스는 스냅샷 제외.
