# ORDER-owner-review-prep-v1 — 운영중(소유주) 축 검토 준비

## 모드: 자율 판정. 멈춤 (a)(b)(c)만.

## 배경
양도인 축 검토 방식(대표 실기기 시뮬레이션 → 버그·설계 수정 누적)을 운영중 축에 반복.
운영중은 등록 객체 없는 스캐폴드 단계 항목 다수 + SalesInputPage 더미 매출 잔존.

## 작업
1. 운영중 온보딩→홈 전수 점검: 암묵 기본값 함정, 뒤로가기·재진입.
2. 운영중 홈 카드 전수 목록화 (카드명/출처/준비중). SalesInputPage 더미 매출 포함 — 제거 금지,
   (가)예시 뱃지 / (나)준비중 교체 구현 지점만 표기.
3. 운영중 "객체" 코드 현황: 내 가게 등록 존재? 양도인 매물 등록과 관계(별도 테이블?부재?).
4. 운영중↔양도인 전환 경로 현황: "양도 준비" 동선이 코드에 있는가.
5. 카피 점검: AI·권위·미구현 약속 grep.
6. docs/SIM-OWNER-v1.md 작성: 신규 이메일 가입→운영중 온보딩→홈→각 카드→(등록 있으면)등록→일정/알림 현황.
