## 0. 먼저 — 오더 전제와 다른 점 2가지

**(1) listings 스키마 변경은 이미 끝나 있습니다.**
`scripts/sql/20260716-industry-category-columns.sql:19-21`이 `listings`에 `category_main`/`category_sub`/`ksic_code` 3컬럼을 이미 추가했고, 실 DB에서도 확인했습니다.

```
listings 조회 결과: {"category_main":null,"category_sub":null,"ksic_code":null}
category_main 채워진 행 수: content-range: */0
```

즉 컬럼은 있고 **전량 NULL**입니다. 이번에 필요한 건 컬럼 추가가 아니라 **쓰는 코드 + 백필**입니다.

## 5. 판단 필요 사항 (대표님)

1. **매핑 불가 7종** 처리 (2-2 ❌ 표) — 특히 `안경` 소분류 신설 여부, 점포 아닌 업종(인력파견·운송·이사) 취급.
