# ORDER-landlord-home-header-card-v1 — 임대인 홈 헤더 집계 + 상가 카드 제목

## 모드: 자율 판정. 멈춤 (a)(b)(c)만.

## 1. 헤더 집계
"상가 N개" → deal_type 집계: "임대 N · 매매 N 진행 중". both는 양쪽 각각 +1(임대2·매매1+both1 → "임대 3 · 매매 2"). 한쪽 0이면 생략("임대 3개 진행 중"). 용어: 사용자 표기 "매매"(매각은 내부, E1p 선택지 통일은 범위 밖).

## 2. 헤더 지역
"지역 미설정 일대" → 실상가 address에서 sidoFromAddress 승격. 여러 시도면 "서울 외 N곳" 축약. 실상가 없으면 A3 폴백.

## 3. 상가 카드 제목
"(상호 없음)" 제거(상가는 상호 없음). 제목 = 행정동 이하 마지막 단위 + 번지("강일동 676-1"). address_detail 호수 있으면 괄호 병기("강일동 676-1 (301호)"). 상세 길면 호수 패턴만, 애매하면 생략(자율). 서브라인 deal_type 표기 유지.

## 테스트: 집계 조합(임대만/매매만/혼합/both)·지역 승격·다지역 축약·카드 제목(호수 유무). 전체.
