# 외출 모드 — 원격 작업 가이드

## 나가기 전 체크리스트 (PC에서)

- [ ] **절전 해제** — `모두외출모드.bat` 실행 시 자동 처리  
  *(수동 확인: 시작 → 전원 및 절전 → 화면/절전 모두 "없음")*
- [ ] **서버 + 터널 시작** — `모두외출모드.bat` 더블클릭 (관리자 권한으로 실행 권장)
- [ ] **비밀번호 확인** — 창에 표시된 내 공개 IP (클립보드에 자동 복사)
- [ ] **창 최소화** — 닫으면 서버 + 터널 종료되므로 최소화만
- [ ] **Claude Code** — 폰에서 claude.ai 열어 이 프로젝트 계속 지시

---

## 외부에서 (폰에서)

### 앱 미리보기
1. 모바일 브라우저에서 **https://modu-dev.loca.lt** 열기
2. 비밀번호 입력 화면이 뜨면 PC에서 복사한 IP 입력 → 확인
3. 이후엔 같은 URL 북마크로 바로 접속

### 코딩 지시
- claude.ai 앱 또는 브라우저 → 대화 이어가기

---

## 고정 URL

| 항목 | 값 |
|------|-----|
| 주소 | `https://modu-dev.loca.lt` |
| 방식 | localtunnel (무료, 계정 불필요) |
| 비밀번호 | 내 공개 IP (bat 실행 시 화면에 표시 + 클립보드 복사) |

> 비밀번호는 내 인터넷 IP가 바뀌면 바뀜 (통상 며칠~수주 유지).  
> bat 실행 때마다 최신 IP를 표시해주므로 별도 확인 불필요.
